    Questions:

    1. The Window Object: as their name said, is a window that contains a DOM (Document Object Model).
      Is possible to open in a browser, and like other objects includes properties and methods.
      Inside the windows are Dialogs, which have the following functions:
      alert() : will pause the execution and display a message
      confirm(), will display a confirmation dialog.
      And prompt(): will show an input field that allows the user to enter text
      Cookies: are small files that are saved locally on the user's computer
      Cookies can only be read by a web page from the same domain
      Cookies are used to personalize the user's browsing experience, keeping track of the user choices
      Cookies are limited to stored 4KB of data
    2. APIs-Application Programming Interface
      Is software that allows the use of the connection between computers or programs
      HTML5 contains a number of APIs that help to get access to hardware, like batteries, geolocation,
      cameras, etc.
      The data Attribute:
      is a way of embedding data in a web page using custom attributes that are ignored by the browser
      Web Workers
      Allows that processes could be run in the background, so the web site can continue to function without fear
      that the page has become "unresponsive"
